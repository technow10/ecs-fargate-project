# ecs-fargate-project
Deploying a dockerized  application on aws using Amazon ECS Fargate.
In this project, we are going to build our infrastructure from scratch.
We will implement this project using - 
* VPC
* Subnets
* Route tables
* Security Groups
* Internet Gateway
* Nat-gateway
* ECS cluster
* Fargate ECS service
* Amazon ECR
* Deployment scripts
